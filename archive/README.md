# Archive

This directory holds historical project artifacts that are no longer part of the active
FinReg baseline but may still be useful for provenance, comparison, or later reference.

Current policy:

- active baseline materials stay in `config/`, `docs/`, `data/domain_finreg/`, `scripts/`, and `src/`
- superseded question-set versions move under `archive/data/`
- superseded reports and planning notes move under `archive/docs/`

Archived files should not be treated as canonical unless they are explicitly promoted back
into the active baseline.
