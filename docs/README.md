# Docs Guide

Start here instead of reading every document chronologically. The repository
contains several research phases, and many older docs are historical notes.

## Current Report Path

- `finreg_final_targeted_benchmark_audit.md`: audit of the final 160-question
  benchmark, strengthened validation checks, and audited metrics.
- `finreg_final_targeted_benchmark_report.md`: concise report-ready explanation
  of the final targeted benchmark and system comparison.
- `poster_right_panel_benchmark_metrics.md`: short poster/deck text for the
  benchmark metrics panel.
- `poster_right_panel_benchmark_metrics.pdf`: one-page poster metrics PDF.
